# Library-Database-Analysis
* Designed and implemented a relational database system to manage library operations, including books, authors, users, and borrowing transactions.

* Created a normalized database schema to ensure data integrity and scalability.

* Wrote complex SQL queries using joins, subqueries, and aggregations to analyze borrowing patterns and book availability.

* Generated insights to support inventory planning and improved user engagement.

* Applied core DBMS concepts such as indexing, relationships, and query optimization.

#### Tech Stack: SQL (MySQL)
#### Concepts: DBMS, Relational Design, Query Optimization
